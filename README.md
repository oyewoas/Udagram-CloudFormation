# Udagram-CloudFormation

## Depoyment

 Deploy the infastructure:

`./create.sh udagraminfra udagram-cfn.yml udagram-cfn-params.json`

 Deploy the servers

`./create.sh udagramservers udagram-servers.yml udagram-servers-params.json`
